# Buscar documentação

## Use quando

- o usuário pedir checagem em docs atuais
- a API/lib parecer recente ou incerta
- o bug parecer dependente de versão

## Não use quando

- a resposta já for estável e conhecida
- a dúvida for genérica e resolvível por inspeção local

## Fontes preferidas

- documentação oficial
- especificações primárias
- OWASP ou MDN para segurança web

## Regra

Registrar claramente quando a resposta foi baseada em documentação atualizada.
